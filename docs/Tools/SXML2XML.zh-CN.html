The YSLib Book

RevisionPatcher

　　位于 Tools/SXML2XML ，用于转换 SXML 文档到 XML 文档。

　　当前只支持 UTF-8 编码，不支持 XML 命名空间。

　　默认使用标准输出打印结果。

构建

　　依赖 YBase 和 YFramework 库，没有特别的构建支持，可以使用 stage 2 SHBuild ，详见 sysroot 安装脚本 。
